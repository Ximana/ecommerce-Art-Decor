Sobre o Projeto

Um sistema revolucionário para diagnóstico assistido de doenças pulmonares

O PulmoVision é um sistema avançado baseado em redes neurais convolucionais (CNN) desenvolvido para auxiliar profissionais de saúde na detecção automática de doenças pulmonares através da análise de imagens de raio-X do tórax.

Deep Learning

Utiliza técnicas avançadas de IA para análise precisa de imagens

Multiplas Doenças

Detecta tuberculose, pneumonia e COVID-19

Alta Precisão

Resultados confiáveis e diagnóstico assistido

Funcionalidades
Upload de Imagens

Carregue facilmente imagens de raio-X para análise

Detecção Automática

Análise rápida e precisa com CNN
